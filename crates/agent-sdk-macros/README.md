# agent-sdk-macros

Optional proc macros for `agent-sdk-toolkit` typed tools. Generated code lowers
into toolkit builders and core tool execution ports; this crate does not own
runtime, policy, journal, event, approval, provider, or storage behavior.

This repository's macro crate is checkout-only for now because the
`agent-sdk-macros` package name is already occupied on crates.io by an
unrelated project.
